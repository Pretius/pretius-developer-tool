Configuration Problem

Pretius Developer Tool is a plugin that exposes many APEX values for the benefit of the Developer.
Whilst the Plugin is only activated whislt the Developer Bar is present, there may be ways that a curious end-user could enable the Devloper Tool thus exposing application logic.
To attain the highest security, it is highly recommended that:
  • You have one and only one Pretius Developer Tool Dynamic True Action on Page Zero
  • The Dynamic Action is associated with a Build Option that is set to Exclude on Export
Whilst the Plugin will still continue to function, this message will nag you until you take the action described above.
